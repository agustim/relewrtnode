PREPARE HARDWARE
----------------

Plug rele in next pins:

	* VCC = 1 or 2
	* GND = 3 or 4
	* GPIO0 = 17

In this side ( http://wrtnode.com/wp-content/uploads/2013/11/jp3.png ) 

Like this example: http://qmp.cat/data/_uploaded/image/wrtnode/qmp-wrt-07.jpg

INSTALL
-------

copy opendoor to /www/cgi-bin


USE
---


from you browser: 

  http://IP_WRTNODE/cgi-bin/opendoor

from command-line:

  wget -qO - http://IP_WRTNODE/cgi-bin/opendoor > /dev/null
